# 專案總覽

本文件說明專案目的、技術選型與檔案結構。

目的：
- 教學為主，讓讀者能理解 Vue3 + TypeScript 的常見應用場景與實作細節。

技術選型（簡述）：
- Vue3 (Composition API)：現代 Vue 開發的首選，可更好地與 TypeScript 結合。
- TypeScript：強型別可提升可維護性與可讀性。
- Vite：快速的開發體驗與極簡建置流程。
- Tailwind CSS：實用 CSS 原子化，方便教學與快速佈局。

主要資料夾：
- `src/`：原始程式碼
- `src/api/`：API 層，統一 axios 管理
- `src/components/`：元件
- `src/router/`：路由設定
- `src/stores/`：Pinia 狀態管理
- `src/views/`：頁面
- `assets/`：圖片與 css
- `docs/`：功能模組說明
- `tests/`：單元測試
